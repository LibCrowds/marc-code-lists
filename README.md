# marc-code-lists

MARC code lists primarily for use in Select2(https://select2.github.io/) select boxes.

Structure of index.json:

```
{
    "countries":[
        {
            "id":"",
            "text":""
        },
        {
            "id":"aa",
            "text":"Albania"
        },
        ...
    ],
    "languages":[
        {
            "id":"",
            "text":""
        },
        {
            "id":"aar",
            "text":"Afar"
        },
        ...
    ],
    "realtors":[
        {
            "id":"",
            "text":""
        },
        {
            "id":"abr",
            "text":"Abridger"
        },
        ...
    ],
    "geographicAreas":[
      {
         "id":"",
         "text":""
      },
      {
         "id":"a",
         "text":"Asia"
      },
    ]
}
```
